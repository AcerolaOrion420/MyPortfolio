~/about

About

I'm currently pursuing my Master's in Computer Applications at BMSIT, after completing my BCA at Kristu Jayanti College — where I also worked as a backend intern building modular, async REST APIs in Java Vert.x for the college's production website, backed by MongoDB and Redis.

What pulls me in these days is the layer just below the application code: how systems stay fast under load, how services get deployed and observed, and how a handful of well-configured boxes can run reliably for months. That curiosity spills over into a home lab I maintain outside of coursework — a small self-hosted server and NAS setup running Arch Linux, where I tinker with reverse proxies, container orchestration, and network configuration for the fun of understanding it end to end.

I'm comfortable moving between Python, Java, and Node.js on the development side, and I'm actively building out my cloud and DevOps toolkit — currently centered on AWS — so I can take systems from "runs on my machine" to properly deployed infrastructure.
